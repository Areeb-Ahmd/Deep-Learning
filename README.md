# Deep Learning

A collection of deep learning projects and tutorials implemented using PyTorch.

## Contents

### PyTorch Basics
- Tensor operations and basics
- Autograd tutorial
- Practice exercises

### Activation Functions
- Implementation and visualization of various activation functions

### Training Neural Networks
- Gradient descent implementations (batch, mini-batch, stochastic)
- Loss functions: binary cross-entropy and categorical cross-entropy
- Digit classification using multilayer perceptron
- Employee bonus prediction
- PyTorch Dataset and DataLoader usage

### Convolutional Neural Networks (CNN)
- CIFAR-10 classification
- Transfer learning
- Data augmentation techniques

### Model Optimization
- Hyperparameter tuning using Optuna
- Regularization techniques:
  - Batch normalization
  - Dropout
  - Early stopping
  - L1 and L2 regularization
- Training algorithms and optimizers comparison

### Hugging Face
- BERT basics and encoding
- BERT spam classification
- GPT-2 next word prediction

## Technologies

- PyTorch
- Torchvision
- Transformers (Hugging Face)
- NumPy
- Pandas
- Matplotlib
- Optuna

## Datasets

Projects use various datasets including:
- MNIST
- Fashion-MNIST
- CIFAR-10
- Sonar dataset
- SMS Spam Collection (UCI)

## Requirements

This repository uses a Python virtual environment. Install dependencies from your virtual environment or requirements file.
